# Together - 일정공유 앱 최적 기획안 (V2)

> **설계 철학**: 실제 가족/팀이 매일 사용하는 패턴 중심

---

## 📊 사용 빈도 분석

### 자주 사용 (일 1회 이상)
```
1순위: 📱 홈 (AI 요약, 오늘 일정/할일 확인)       → 70%
2순위: 📅 일정 (전체 일정 확인)                  → 60%
3순위: ✓ 할일 (진행 상황 확인)                   → 55%
4순위: 🔔 알림 (새로운 항목 확인)               → 50%
```

### 가끔 사용 (주 1회)
```
- ⚙️ 그룹 설정/전환                              → 30%
- 👤 프로필 관리                                  → 15%
```

### 거의 안 함 (월 1회 이하)
```
- 👥 초대하기 (한 번 초대 후 거의 안 함)        → 5%
```

## 🎨 최적 화면 구조 (재설계)

### 헤더 (상단 고정)
```
┌──────────────────────────────────────────────┐
│ ≡ (그룹/프로필)  Together              🔔 ⋮  │
│                 (우리가족)             (알림)(메뉴) │
└──────────────────────────────────────────────┘
```

**좌측**:
- 그룹 전환 / 프로필 메뉴
- 그룹 선택 → 그룹 목록 사이드슬라이더

**중앙**:
- 현재 그룹명 표시

**우측**:
- 🔔 **알림 벨** (빨간 뱃지: 새로운 항목 수)
- ⋮ **메뉴** (설정, 초대, 나가기)

---

## 📱 메인 화면 (하단 탭)

### 하단 탭 구조 (3개만)
```
┌──────────────────────────────────────────────┐
│                                              │
│            [탭별 콘텐츠]                    │
│                                              │
├──────────────────────────────────────────────┤
│  🏠 홈      │      📅 일정      │      ✓ 할일    │
└──────────────────────────────────────────────┘
```

**왜 3개?**
- 초대는 헤더 메뉴(⋮)로 이동
- 사용자는 일정/할일 추가 후 거의 초대를 안 함
- 핵심 3개만 하단에 → 더 큰 터치 영역

## 🎯 중앙 FAB 버튼 (부동 액션 버튼)

### 위치: 탭 위, 화면 중앙 하단

```
                        ┌─────────────┐
                        │ + 일정 추가  │ → 일정 작성 폼
                        └─────────────┘
                              ▲
                        ┌─────┴─────┐
                        │     +      │
                        │  (메인)    │
                        └─────┬─────┘
                              ▼
                        ┌─────────────┐
                        │ + 할일 추가  │ → 할일 작성 폼
                        └─────────────┘
```

**기능:**
- ✅ 탭 간 이동 없이 빠르게 추가
- ✅ 일정/할일 선택지 제시
- ✅ 모든 탭에서 접근 가능

## 🔔 알림 시스템

### 알림 트리거
```
1️⃣ 구성원이 일정 추가
   → 푸시: "[라면]이 출장 일정을 추가했습니다"
   → 홈 피드에 표시
   
2️⃣ 구성원이 할일 추가
   → 푸시: "[민준]이 회의 자료 작성 할일을 추가했습니다"
   → 홈 피드에 표시
   
3️⃣ 할일 마감 임박 (24시간 전)
   → 푸시: "회의 자료 작성이 내일 까지입니다"
   → 할일 탭에서 강조

4️⃣ 일정 시작 1시간 전
   → 푸시: "팀 회의가 1시간 뒤에 시작합니다"
   → 화면 상단 배너

5️⃣ 구성원이 온라인/오프라인
   → 홈 화면 멤버 상태 표시
```

### 알림 벨 UI
```
┌─────────┐
│🔔  ⓓ 3 │  ← 빨간 뱃지: 읽지 않은 알림 수
└─────────┘

클릭 → 알림 목록 모달
```
