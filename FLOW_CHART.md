## 🎯 각 화면 상세 설명

### 1. 로그인 화면 (Login Screen)
```
┌──────────────────────────────────┐
│                                  │
│         Together                 │
│  가족의 일정을 함께 관리하세요    │
│                                  │
│    [Google 로그인]               │
│                                  │
└──────────────────────────────────┘
```
- **기능**: Google OAuth 인증
- **다음 화면**: 그룹 선택 (기존 그룹이 있으면) 또는 새 그룹 생성

## 🎯 그룹 타입별 특성 & 활용 방식

| 타입 | 용도 | 주요 기능 | 예시 |
|------|------|---------|------|
| 👨‍👩‍👧‍👦 **가족** | 가족 일정 관리 | 가족 행사, 약속, 할일 공유 | 우리가족, 친가족 |
| 👔 **프로젝트** | 팀 프로젝트 진행 | 마감일, 태스크, 회의 일정 | 개발팀, 마케팅팀 |
| 🎯 **동호회** | 취미 커뮤니티 관리 | 모임 일정, 행사, 스터디 | 독서클럽, 운동모임 |
| 💑 **커플** | 연인 일정 공유 | 데이트, 기념일, 공동 계획 | 우리 둘 |
| 📌 **기타** | 기타 목적 | 자유롭게 사용 | 프로젝트, 모임 등 |

## ⚙️ 추가 기능 (향후)

```
추가 옵션 메뉴 (3점 버튼)
  ├─ 그룹 설정
  │   ├─ 그룹 이름 수정
  │   ├─ 그룹 설명 추가

  ├─ 멤버 관리

  │   ├─ 멤버 강제 퇴장
  │   └─ 멤버 목록
  │
  └─ 위험 구역
      ├─ 그룹 나가기
      └─ 그룹 삭제 (소유자만)
```
